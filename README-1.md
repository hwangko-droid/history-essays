# 이름이 아니다 — 다시 읽는 한국 고대사

> **신화를 공유하는 자들의 역사.** 전 21편으로, 한국 고대사를 '한 민족의 역사'가 아니라 '신화를 공유한 자들의 이야기'로 다시 읽는 시도입니다.

이 저장소에는 같은 내용의 **두 가지 판본**이 있습니다.

## 이 책의 세 가지 큰 명제

1. **민족은 없다, 신화의 공유가 있을 뿐이다.**
2. **성에 대한 도덕은 상속의 그림자다.**
3. **국경은 없다, 거점과 길의 그물이 있을 뿐이다.**

> 이 책의 어원학은 영토를 소급하지 않고, 신화학은 혈통을 소급하지 않는다. 추적하는 것은 오직 의미의 상실과, 이야기의 공유다. 위서(환단고기·규원사화 등)는 논거로 채택하지 않으며, 현대의 어떤 영토·민족 주장과도 무관하다.

### 🔧 전자책 링크를 켜는 법 (GitHub Pages)

1. 이 저장소 상단 **Settings** → 왼쪽 메뉴 **Pages**
2. **Source**에서 **Deploy from a branch** 선택
3. **Branch**를 `main` / `/ (root)` 으로 두고 **Save**
4. 1~2분 뒤 `https://hwangko-droid.github.io/history-essays/` 주소가 활성화됩니다.
